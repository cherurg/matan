3. Предел числовой последовательности

Числовые последовательности

Отображение $\nat \mapsto X$ будем называть последовательностью и записывать как $x_1, x_2,\dots, x_n$. Отображение $\nat \mapsto \real$ будем называть числовой последовательностью.

Числовая последовательность называется ограниченной сверху (снизу), если $\exists M \in \real\; (m \in \real): \forall n \in \nat \Rightarrow x_n \le M\; (x_n \ge m)$. Ч.п. $\{x_n\}$ ограничена, если $\exists A \in \real: \forall n \in \nat \Rightarrow |x_n| \le A$.

Ч.п. $\{x_n\}$ ограничена тогда и только тогда, когда $\{x_n\}$ ограничена сверху и снизу. Ч.п. $\{x_n\}$ называется неограниченной, если $\forall A \in \real \; \exists n \in \nat : |x_n| > A$.

$\{x_n\}$ называется бесконечно большой, если $ \forall A \in \real \; \exists N(A): \forall n \ge N \Rightarrow |x_n| > A$. Всякая бесконечно большая числовая последовательность является неограниченной, но обратное, вообще говоря, неверно. Например, рассмотрим следующую неограниченную последовательность: $$x_n = \frac{n+(-1)^{n+1}\cdot n}{2}.$$ Докажем, что $\exists A_0 \in \real : \forall N \; \exists n_0 \ge N \Rightarrow |x_{n_0}| < A_0$ (отрицание определения б.б.ч.п.). Можно предъявить $A_0 = 1: \forall N \in \nat \; \exists k_0 \ge N : |x_{2k_0}| = 0<1=A_0$.

$\{x_n\}$ называется бесконечно малой, если $ \forall \eps >0 \; \exists N(\eps): \forall n \ge N(\eps) \Rightarrow |x_n| < \eps$.

Выберете последовательность:
$x_n = \left\{\frac 12,\frac 13,\frac 23,\frac 14,\frac 24,\frac 34,\dots\right\}$ $x_n = (-1)^n\cdot \left(1 + \frac 1n\right)$
$x_n = (-1)^n\cdot \left(1+\frac 1n\right)^n$ $x_n = \sin n$
$\left\{ \begin{aligned} x_n &= \left(1+\frac 1n\right)^n\\ y_n &= \sum_{i=0}^n \frac{1}{i!} \end{aligned} \right.$
$x_n = \left\{\frac 12,\frac 12,\frac 13,\frac 12,\frac 13,\frac 14,\dots\right\}$
$x_n = \left\{\frac 11-\frac 12,\frac 11-\frac 13,\frac 12-\frac 14,\frac 11-\frac 15,\frac 12-\frac 16,\frac 13-\frac 17,\dots\right\}$
$x_n = \left\{0, 1, -1 , 0, 1, -1, 0, 1, -1, 2, -2, 0, 1, -1, 2, -2, 3, -3,\dots\right\}$ $x_n = \frac{2+(-1)^n}{n}$
$x_n = \frac{1+(-1)^n}{n}$ $x_n = (-1)^n$
$x_n = (0.9)^n$ $x_n = \frac{(-1)^n}{n}$

Свойства бесконечно малых числовых последовательностей

Пусть $\term \alpha$ — бесконечно малая числовая последовательность.
  1. $\term \alpha$ ограничена.
    Как известно, $\forall \eps>0 \; \exists N(\eps): \forall n \ge N \Rightarrow |\alpha_n| < \eps$. Значит, для всех $n \ge N$ доказано. Но $\forall n < N \Rightarrow \alpha_n \le \max\{|\alpha_1|,|\alpha_2|,\dots,|\alpha_{N-1}|\}$. Тогда выберем $\eps = 1,\, A = \max\{|\alpha_1|,\dots,|\alpha_{N-1}|,1\} \Rightarrow \forall n \in \nat,\, |\alpha_n| \le A$.
  2. Если $\{y_n\}$ ограничена, то $\{\alpha_n\cdot y_n\}$ — бесконечно малая.
    $\{\alpha_n\}$ — бесконечно малая, поэтому $\forall \eps >0 \; \exists N(\eps): \forall n \ge N(\eps) \Rightarrow |\alpha_n| < \frac{\eps}{A}$. Ввиду ограниченности $\{y_n\}$, $\exists A: \forall n \in \nat \Rightarrow |y_n| \le A$. Но тогда $\{\alpha_n \cdot y_n\}: \forall \eps >0 \; \exists N(\eps): \forall n \ge N \Rightarrow |\alpha_n \cdot y_n| < \frac{\eps}{A} \cdot A = \eps$.
  3. Если $\term \beta$ — бесконечно малая, то $\{\alpha_n \pm \beta_n\} \text{ и } \{\alpha_n \cdot \beta_n\}$ — бесконечно малые.
    \begin{gather*} \forall \eps >0 \; \exists N_1(\eps): \forall n \ge N_1 \Rightarrow |\alpha_n| < \frac{\eps}{2} \text{ и } \exists N_2(\eps): \forall n \ge N_2 \Rightarrow |\beta_n| < \frac{\eps}{2}.\\ \text{Тогда при } N = \max\{N_1,N_2\} \Rightarrow \forall n \ge N \Rightarrow |\alpha_n \pm \beta_n| \le |\alpha_n| + |\beta_n| < \frac{\eps}{2} + \frac{\eps}{2} = \eps. \end{gather*} Для произведения получаем аналогично: \begin{gather*} \forall \eps >0 \; \exists N_1(\eps): \forall n \ge N_1 \Rightarrow |\alpha_n| < \frac{1}{\eps} \text{ и } \exists N_2(\eps): \forall n \ge N_2 \Rightarrow |\beta_n| < \eps^2.\\ \text{Тогда при } N = \max\{N_1,N_2\} \Rightarrow \forall n \ge N \Rightarrow |\alpha_n \cdot \beta_n| = |\alpha_n| \cdot |\beta_n| < \frac{1}{\eps} \cdot \eps^2 = \eps. \end{gather*}

Связь бесконечно больших и бесконечно малых последовательностей

Теорема 3.1 Если $\{x_n\}$ — бесконечно малая и $\forall n \in \nat \Rightarrow x_n \ne 0$, то $\left\{\frac{1}{x_n}\right\}$ — бесконечно большая.
$\forall \eps >0 \; \exists N(\eps): \forall n \ge N \Rightarrow |x_n| < \eps \Leftrightarrow \frac{1}{|x_n|} > \frac{1}{\eps} = A$.
Теорема 3.2 Если $\{x_n\}$ — бесконечно большая и $\forall n \in \nat \Rightarrow x_n \ne 0$, то $\left\{\frac{1}{x_n}\right\}$ — бесконечно малая.
$\forall A >0 \; \exists N(A): \forall n \ge N \Rightarrow |x_n| > A \Leftrightarrow \frac{1}{|x_n|} < \frac{1}{A} = \eps$.

Предел числовой последовательности

Назовем окрестностью точки $a \in \real$ любой интервал, содержащий эту точку, и будем обозначать ее как $\mathrm U(a)$. $\mathrm U_\eps(a) = (a-\eps; a+\eps)$ — симметричная $\eps$-окрестность. $\mathrm U'_\eps(a) = (a-\eps;a) \cup (a;a+\eps) = \overset{0}{\mathrm U}_\eps = \mathrm U_\eps \setminus \{a\}$ — проколотая. Будем говорить, что числовая последовательность $\{x_n\}$ имеет предел, равный $a$, если \begin{align} \cllim x_n &= a \Leftrightarrow \forall \mathrm U_\eps(a) \; \exists N(\mathrm U_\eps(a)): \forall n \ge N \Rightarrow x_n \in \mathrm U_\eps(a).\\ \cllim x_n &= a \Leftrightarrow \forall \eps >0 \; \exists N(\eps): \forall n \ge N \Rightarrow |x_n - a| < \eps. \end{align}
Теорема 3.3 Оба определения предела числовой последовательности эквивалентны.

Рассмотрим 1-ое определение. Если $x_n \in \mathrm U_\eps(a)$, то $a-\eps < x_n < a+\eps \Leftrightarrow -\eps < x_n - a < \eps \Rightarrow |x_n - a| < \eps$.

Теперь рассмотрим 2-ое определение. Если $|x_n - a| < \eps$, то $a-\eps < x_n < a+\eps$, т.е. $x_n \in \mathrm U_\eps(a)$.
Заметим, что оба определения предела фактически говорят о том, что последовательность $\{x_n-a\}$ — бесконечно малая. Если существует $\cllim x_n = a <\infty$, то $\{x_n\}$ — сходящаяся последовательность. Иначе, если не существует $\cllim x_n \text{ или } \cllim x_n = \infty$, то $\{x_n\}$ — расходящаяся. Используя отрицание определения: $\{x_n\}$ — расходится $\Leftrightarrow \forall a \in \real \; \exists \eps_0 > 0: \forall N \in \nat \; \exists n_0 \ge N: |x_n - a| \ge \eps_0.$
