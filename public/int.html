Задача: вычислить интеграл \({\Large \int_{}^{}}\!\sqrt{1-\sin 2x}dx\).
Одна из систем символьных вычислений предлагает такой ответ: \({\large \int_{}^{}}\!\sqrt{1-sin(2x)}dx={\large -{\frac { \left( -1+\sin \left( 2\,x \right) \right) \left( \sin \left( 2\,x \right) +1 \right) }{\cos \left( 2\,x \right) \sqrt {1- \sin \left( 2\,x \right) }}}}+C\)

Но этот ответ не является правильным. И действительно, рассмотрим функцию \(f(x)=\sqrt{1-\sin 2x}=\sqrt{(\cos x - \sin x)^2}=|\cos x-\sin x|=(\cos x-\sin x)\text{sgn}(\cos x - \sin x) \). Обозначив за \(I(x)={\Large \int_{}^{}}\!\sqrt{1-\sin 2x}dx \), находим
\[I(x)=\begin{cases}\qquad...............\\-(\sin x - \cos x)+C_{-1}, & \pi/4-2\pi\leq x<\pi/4-\pi\\ \sin x + \cos x +C_{0}, &\pi/4-\pi\leq x<\pi/4 \\ -(\sin x - \cos x)+C_{1}, &\pi/4\leq x<\pi/4+\pi \\\qquad...............\\ (-1)^n(\sin x + \cos x)+C_n,& \pi/4+(n-1)\pi\leq x< \pi/4 + n\pi \\ \qquad...............\end{cases}\] Поскольку первообразная непрерывна, то должно выполняться равенство \[I(\pi/4+k\pi)=I(\pi/4+k\pi-0),\; k\in\mathbb{Z} \] т.е. \[(-1)^{k+1}(\sin x_k + \cos x_k)+C_{k+1}=\lim \limits_{x \to x_k-0}(-1)^k(\sin x + \cos x)+C_k, \text{ где } x_k=\pi/4+k\pi, \: k\in \mathbb{Z}\] Отсюда приходим к равенству \(-\sqrt{2}+C_{k+1}=\sqrt{2}+C_k\). При \(k=0\) находим \(C_1=2\sqrt{2}+C_0\); далее, при \(k=1\) получаем \(C_2=2\sqrt{2}+C_1=2\cdot2\sqrt{2}+C_0\). С помощью метода математической индукции устанавливаем, что \(C_n=2\sqrt{2}n+C\), где \(C=C_0\) -- произвольная постоянная.
Наконец, преобразуя неравенство \(\pi/4+(n-1)\pi\leq x< \pi/4+n\pi\) к виду \[n \leq \frac{x-\pi/4+\pi}{\pi}< n+1, \] находим, что \[n=\left[ \frac{x-\pi/4+\pi}{\pi} \right].\] Таким образом, \[{\large \int_{}^{}}\!\sqrt{1-\sin 2x}dx= (-1)^{{\large \left[ \frac{x-\pi/4+\pi}{\pi} \right]}}(\sin x + \cos x)+2\sqrt{2}\left[ \frac{x-\pi/4+\pi}{\pi} \right]+C.\] График правильного ответа выглядит так (здесь \(C=0\)):
